# 터지지 않은 폭탄쥐 연속 유언 테스트 수정

`src/shared/sof-rules.test.ts`의 연속 유언 테스트가 폭탄쥐를 실제로 죽이지 못하던 문제를 수정했습니다.

기존 픽스처는 `temporaryHealthModifier: -1`만 부여했습니다. 턴 종료 시 임시 체력 변화가 제거되면 폭탄쥐의 체력이 0에서 기본 체력 1로 회복되므로 유언이 생성되지 않았습니다.

수정된 픽스처는 폭탄쥐마다 다음 상태를 사용합니다.

- 누적 피해 1
- 임시 체력 +1

턴 종료 전에는 체력 2에 피해 1로 살아 있고, 턴 종료 시 임시 체력 +1이 사라지면 체력 1에 피해 1이 되어 두 장이 동시에 묘지로 갑니다.

규칙 엔진 직접 실행 결과:

- 첫 선택 대기: 폭탄쥐 유언 2개
- 첫 유언으로 상대 몬스터 제거
- 두 번째 유언: 대상 없음 상태에서 빈 선택으로 종료
- 최종 선택 대기: 0개
