# 카드 프레임 이미지 자리

`card-frame.png`을 같은 이름으로 덮어쓰면 덱 빌더, 손패, 전장, 마나, 카드 상세 보기에 자동으로 적용됩니다.

## 권장 규격

- 파일: `public/ui/card-frames/card-frame.png`
- 형식: 투명 PNG
- 비율: 정확히 5:7
- 권장 크기: 1000×1400 또는 1500×2100
- 중앙 일러스트 영역과 이름·수치가 보일 영역은 투명해야 합니다.
- 프레임 그림 안에는 카드명, 비용, 속성, 공격력, 체력을 직접 넣지 마세요. 게임 UI가 별도 레이어로 출력합니다.
- 네 모서리 배지는 프레임에 그려도 되지만 실제 숫자와 속성 글자를 가리지 않도록 충분히 비워두세요.

## 미세 조정

`src/ui-overhaul.css`의 다음 값을 바꾸면 됩니다.

```css
--card-frame-inset: 0px;
--card-frame-opacity: 1;
--card-frame-size: 100% 100%;
```

프레임 가장자리에 투명 여백이 있다면 `--card-frame-inset`을 음수로, 너무 크게 보이면 양수로 조정하세요.
