# Duel Spirits 폰트 자리

이 폴더에 아래 두 WOFF2 파일을 넣으면 전 UI에 자동으로 적용됩니다.

- `public/ui/fonts/ui.woff2`
  - 본문, 버튼, 숫자, 필터, 상세 설명용
  - 한글과 영문을 모두 포함한 가변 폰트를 권장합니다.
- `public/ui/fonts/display.woff2`
  - 카드 이름, 화면 제목, 상태 바, 턴 종료 버튼용
  - 한글 카드명이 한 줄로 읽혀야 하므로 지나치게 넓거나 장식적인 폰트는 피하세요.

파일이 없으면 기존 시스템 폰트로 자동 대체되므로 배포는 깨지지 않습니다.

## 권장 조건

- 형식: WOFF2
- 한글 완성형 글리프 포함
- `ui.woff2`: 100~900 가변 굵기 권장
- `display.woff2`: 400~900 가변 굵기 권장
- 라이선스와 게임 내 상업 이용 가능 여부를 반드시 확인하세요.

폰트 파일 자체는 저장소 용량이 커질 수 있으므로, 최종 후보가 확정되기 전에는 Git LFS나 별도 에셋 관리도 고려할 수 있습니다.
