# v7.1 test fixture correction

`마지막 불씨`의 개정 체력은 1입니다. 기존 고립 동작 검사는 공격력 1인 `나무에 사는 요정`과 전투시켜 놓고 공격 후에도 `마지막 불씨`가 전장에 남아 있다고 기대하여 실패했습니다.

규칙 구현은 그대로 두고, 고립 공격력의 즉시 활성화만 검증하도록 첫 수비자의 임시 공격력 보정을 -1로 설정했습니다. 이제 해당 수비자는 공격력 0, 충분한 체력을 가지므로:

- 고립 상태의 마지막 불씨가 피해 3을 줌
- 마지막 불씨는 생존하고 공격 횟수/소진 상태를 검사할 수 있음
- 비고립 상태에서는 피해 2를 줌

을 각각 독립적으로 검증합니다.
